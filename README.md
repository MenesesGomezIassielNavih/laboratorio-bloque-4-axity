## Contexto del Bloque 4

El Bloque 4 cubre los siguientes ejes tecnicos segun el plan curricular del programa:

- **Modulo 21 - Java Spring Boot:** Spring MVC, controladores REST, GraphQL, Swagger y pruebas de endpoints. Instructor: Alan Martinez (JAMNDEV).
- **Modulo 22 - Java Spring Batch:** procesamiento masivo de datos, Jobs, Steps, Readers, Processors y Writers con chunks. Instructor: Alan Martinez (JAMNDEV).
- **Modulo 23 - TDD + Pruebas Unitarias:** Test-Driven Development con ciclo Rojo-Verde-Refactorizar, JUnit 5 y Mockito. Instructor: Noel Telesforo Martinez.
- **Modulo 24 - Angular + Unit Test:** Componentes, servicios, Reactive Forms, Guards, Interceptors y testing con Jasmine, Karma y TestBed. Instructor: Jose Williams Alvarado.
- **Modulo 25 - JWT y OAuth2:** Fundamentos de autenticacion, JSON Web Tokens, OAuth 2.0 y OpenID Connect. Instructor: Luis Antonio Hernandez.
